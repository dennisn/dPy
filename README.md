# dPy
Dennis Python snippets

## Installation

- Install python (https://www.python.org/downloads/)
- To install the requirements in the current virutal env: `python -m pip install -r requirements.txt`
  - Activate virtual env.: `venv/Scripts/activate.ps1`
    - on Linux/WSL: `source venv/Scripts/activate`

## Tools

- Selenium spider: `python scripts\SeleniumSpider\download_123truyen_Seleniumbase.py`
  - Parameter is internally set in the script
- Youtube audio download: `python scripts\Youtube\DownloadPlaylistAudio.py`
  - Enter the playlist URL
  - Downloaded audio will be saved as MP3 in folder with the same name as the playlist
